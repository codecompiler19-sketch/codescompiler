An HTML iframe is used to display a web page within a web page.

### Iframe Syntax
The HTML `<iframe>` tag specifies an inline frame.

An inline frame is used to embed another document within the current HTML document.

<Editor 
    initialHtml={`<h2>HTML Iframes</h2>\n<p>You can use the height and width attributes to specify the size of the iframe:</p>\n\n<iframe src="https://www.example.com" width="100%" height="300" title="Example Website"></iframe>`}
/>

### Removing the Border
By default, an iframe has a border around it. To remove the border, add the `style` attribute and write CSS `border: none;`.

<Editor 
    initialHtml={`<h2>Removing the Iframe Border</h2>\n<p>Add style="border:none;" to remove the border:</p>\n\n<iframe src="https://www.example.com" style="border:none;" width="100%" height="200" title="Example Website"></iframe>`}
/>

### Iframe as a Target for a Link
An iframe can be used as the target frame for a link. The `target` attribute of the link must refer to the `name` attribute of the iframe.

<Editor 
    initialHtml={`<h2>Iframe Target Link</h2>\n\n<iframe src="https://www.example.com" name="iframe_a" width="100%" height="200" title="Iframe Example"></iframe>\n<br/>\n\n<p><a href="https://example.org" target="iframe_a">Click here to open a new page in the iframe!</a></p>`}
/>
