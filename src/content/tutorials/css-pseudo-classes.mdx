A pseudo-class is used to define a special state of an element.

For example, it can be used to:
*   Style an element when a user mouses over it
*   Style visited and unvisited links differently
*   Style an element when it gets focus

### Syntax
The syntax of pseudo-classes:

```css
selector:pseudo-class {
  property: value;
}
```

### Anchor Pseudo-classes
Links can be styled differently depending on what state they are in.

<Editor 
    initialHtml={`<a href="https://example.com" target="_blank">This is a link</a>`}
    initialCss={`/* unvisited link */\na:link {\n  color: #FF0000;\n}\n\n/* visited link */\na:visited {\n  color: #00FF00;\n}\n\n/* mouse over link */\na:hover {\n  color: #FF00FF;\n}\n\n/* selected link */\na:active {\n  color: #0000FF;\n}`}
/>

### The :hover Pseudo-class on `<div>`
You can use the `:hover` pseudo-class on all elements, not only on links.

<Editor 
    initialHtml={`<div class="testHover">Mouse Over Me</div>`}
    initialCss={`.testHover {\n  background-color: green;\n  color: white;\n  padding: 25px;\n  text-align: center;\n  transition: 0.3s;\n}\n\n.testHover:hover {\n  background-color: blue;\n}`}
/>

### The :first-child Pseudo-class
The `:first-child` pseudo-class matches a specified element that is the first child of another element.

<Editor 
    initialHtml={`<div>\n  <p>This paragraph is the first child of its parent (div).</p>\n  <p>This paragraph is not the first child.</p>\n</div>`}
    initialCss={`p:first-child {\n  color: blue;\n}`}
/>
