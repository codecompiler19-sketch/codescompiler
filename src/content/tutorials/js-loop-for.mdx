Loops can execute a block of code a number of times. Loops are handy if you want to run the same code over and over again, each time with a different value.

### The For Loop
The `for` loop has the following syntax:

```javascript
for (statement 1; statement 2; statement 3) {
  // code block to be executed
}
```

*   **Statement 1:** is executed (one time) before the execution of the code block.
*   **Statement 2:** defines the condition for executing the code block.
*   **Statement 3:** is executed (every time) after the code block has been executed.

### Looping Over an Array
Often, a `for` loop is used to iterate over the items inside an array.

<Editor 
    initialHtml={`<h2>Looping through an Array</h2>\n<p id="demo"></p>`}
    initialJs={`const cars = ["BMW", "Volvo", "Saab", "Ford", "Fiat", "Audi"];\nlet text = "";\n\nfor (let i = 0; i < cars.length; i++) {\n  text += cars[i] + "<br>";\n}\n\ndocument.getElementById("demo").innerHTML = text;`}
/>

### The For...In Loop
The JavaScript `for...in` statement loops through the properties of an Object.
