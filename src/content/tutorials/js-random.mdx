`Math.random()` returns a random number between 0 (inclusive),  and 1 (exclusive).

### Math.random()
`Math.random()` always returns a number lower than 1.

<Editor 
    initialHtml={`<h2>JavaScript Math.random()</h2>\n<p id="demo"></p>`}
    initialJs={`document.getElementById("demo").innerHTML = Math.random();`}
/>

### JavaScript Random Integers
`Math.random()` used with `Math.floor()` can be used to return random integers.

<Editor 
    initialHtml={`<h2>JavaScript Random Integers</h2>\n<p>Returns a random integer from 0 to 9:</p>\n<p id="demo"></p>`}
    initialJs={`document.getElementById("demo").innerHTML = Math.floor(Math.random() * 10);`}
/>

By multiplying Math.random() with 10, the result is between 0 and 9.9999. Then, Math.floor() rounds it down to the nearest integer.

<Editor 
    initialHtml={`<h2>Random 1 to 10</h2>\n<p>Returns a random integer from 1 to 10:</p>\n<p id="demo"></p>`}
    initialJs={`document.getElementById("demo").innerHTML = Math.floor(Math.random() * 10) + 1;`}
/>

### A Proper Random Function
As you can see from the examples above, it might be a good idea to create a proper random function to use for all random integer purposes.

This JavaScript function always returns a random number between min (included) and max (excluded):

<Editor 
    initialHtml={`<h2>Proper Random Function</h2>\n<button onclick="document.getElementById('demo').innerHTML = getRndInteger(0,10)">Click Me</button>\n<p id="demo"></p>`}
    initialJs={`function getRndInteger(min, max) {\n  return Math.floor(Math.random() * (max - min) ) + min;\n}`}
/>
