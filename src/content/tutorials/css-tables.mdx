The look of an HTML table can be greatly improved with CSS.

### Table Borders
To specify table borders in CSS, use the `border` property.

<Editor 
    initialHtml={`<table>\n  <tr>\n    <th>Company</th>\n    <th>Contact</th>\n    <th>Country</th>\n  </tr>\n  <tr>\n    <td>Alfreds Futterkiste</td>\n    <td>Maria Anders</td>\n    <td>Germany</td>\n  </tr>\n  <tr>\n    <td>Centro comercial Moctezuma</td>\n    <td>Francisco Chang</td>\n    <td>Mexico</td>\n  </tr>\n</table>`}
    initialCss={`table, th, td {\n  border: 1px solid black;\n}`}
/>

Notice that the table in the example above has double borders. This is because both the table and the `<th>` and `<td>` elements have separate borders.

### Collapse Table Borders
The `border-collapse` property sets whether the table borders should be collapsed into a single border.

<Editor 
    initialHtml={`<table>\n  <tr>\n    <th>Company</th>\n    <th>Contact</th>\n  </tr>\n  <tr>\n    <td>Alfreds Futterkiste</td>\n    <td>Maria Anders</td>\n  </tr>\n</table>`}
    initialCss={`table {\n  border-collapse: collapse;\n  width: 100%;\n}\n\ntable, th, td {\n  border: 1px solid black;\n}`}
/>

### Table Padding, Width, and Alignment
We can make tables look fantastic just by using padding, proper text alignment, and background colors on headings.

<Editor 
    initialHtml={`<table>\n  <tr>\n    <th>Company</th>\n    <th>Contact</th>\n    <th>Country</th>\n  </tr>\n  <tr>\n    <td>Alfreds Futterkiste</td>\n    <td>Maria Anders</td>\n    <td>Germany</td>\n  </tr>\n  <tr>\n    <td>Centro comercial Moctezuma</td>\n    <td>Francisco Chang</td>\n    <td>Mexico</td>\n  </tr>\n  <tr>\n    <td>Ernst Handel</td>\n    <td>Roland Mendel</td>\n    <td>Austria</td>\n  </tr>\n</table>`}
    initialCss={`table {\n  border-collapse: collapse;\n  width: 100%;\n  font-family: sans-serif;\n}\n\nth, td {\n  text-align: left;\n  padding: 12px;\n}\n\ntr:nth-child(even) {background-color: #f2f2f2;}\n\nth {\n  background-color: #04AA6D;\n  color: white;\n}`}
/>
