CSS has a lot of properties for formatting text layout, alignment, and decor.

### Text Color
The `color` property is used to set the color of the text.

<Editor 
    initialHtml={`<h1>This is a Heading</h1>\n<p>This is an ordinary paragraph. Notice that this text is blue. The default text color for a page is defined in the body selector.</p>`}
    initialCss={`body {\n  color: blue;\n}\nh1 {\n  color: green;\n}`}
/>

### Text Alignment
The `text-align` property is used to set the horizontal alignment of a text. A text can be left or right aligned, centered, or justified.

<Editor 
    initialHtml={`<h1 class="center">Centered Heading</h1>\n<p class="left">This text is aligned to the left.</p>\n<p class="right">This text is aligned to the right.</p>`}
    initialCss={`h1.center { text-align: center; }\np.left { text-align: left; }\np.right { text-align: right; }`}
/>

### Text Decoration
The `text-decoration` property is used to set or remove decorations from text. The value `text-decoration: none;` is often used to remove underlines from links.

<Editor 
    initialHtml={`<a href="#">This is a link without an underline</a>\n<br><br>\n<p class="overline">This text has a line above it.</p>\n<p class="line-through">This text has a line through it.</p>\n<p class="underline">This text has a line under it.</p>`}
    initialCss={`a { text-decoration: none; }\np.overline { text-decoration: overline; }\np.line-through { text-decoration: line-through; }\np.underline { text-decoration: underline; }`}
/>

### Text Transformation
The `text-transform` property is used to specify uppercase and lowercase letters in a text.

<Editor 
    initialHtml={`<p class="uppercase">This text is transformed to uppercase.</p>\n<p class="lowercase">This text is transformed to lowercase.</p>\n<p class="capitalize">This text is capitalized.</p>`}
    initialCss={`p.uppercase { text-transform: uppercase; }\np.lowercase { text-transform: lowercase; }\np.capitalize { text-transform: capitalize; }`}
/>
