Input elements can be heavily customized to provide validation, placeholders, and structure.

### The placeholder Attribute
The `placeholder` attribute specifies a hint that describes the expected value of an input field.

<Editor 
    initialHtml={`<h2>Placeholder Attribute</h2>\n<form>\n  <input type="text" name="fname" placeholder="First Name">\n  <br><br>\n  <input type="email" name="email" placeholder="john@example.com">\n</form>`}
/>

### The required Attribute
The `required` attribute specifies that an input field must be filled out before submitting the form.

<Editor 
    initialHtml={`<h2>Required Attribute</h2>\n<form>\n  Username: <input type="text" name="usrname" required>\n  <input type="submit">\n</form>`}
/>

### The disabled Attribute
The `disabled` attribute specifies that an input field is disabled (unusable and un-clickable).

<Editor 
    initialHtml={`<h2>Disabled Attribute</h2>\n<form>\n  First name: <input type="text" name="fname" value="John" disabled>\n</form>`}
/>
