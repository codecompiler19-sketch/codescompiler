HTML `<svg>` defines vector-based graphics in XML format.

### What is SVG?
* SVG stands for Scalable Vector Graphics
* SVG is used to define graphics for the Web
* SVG images can be scaled up or down without losing quality

### Drawing a Circle
Here is an example of a simple SVG graphic.

<Editor 
    initialHtml={`<h2>SVG Circle</h2>\n\n<svg width="100" height="100">\n  <circle cx="50" cy="50" r="40" stroke="green" stroke-width="4" fill="yellow" />\n  Sorry, your browser does not support inline SVG.\n</svg>`}
/>
