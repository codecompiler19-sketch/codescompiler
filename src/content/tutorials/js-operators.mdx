### Operator Quick Reference

| Type | Operators | Example |
| :--- | :--- | :--- |
| Arithmetic | `+ - * / % **` | `10 % 3 → 1` |
| Assignment | `= += -= *= /=` | `x += 5` |
| Comparison | `== === != !== > < >= <=` | `5 === "5" → false` |
| Logical | `&& \|\| !` | `true && false → false` |
| String | `+` | `"Hi" + " there"` |

> **Tip:** Always use `===` (strict equality) instead of `==`. It checks both **value** AND **type**, preventing sneaky `"5" == 5` bugs.
