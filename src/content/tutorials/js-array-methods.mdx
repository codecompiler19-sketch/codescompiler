Arrays provide a lot of built-in methods.

### JavaScript Array length
The `length` property returns the length (size) of an array.

<Editor 
    initialHtml={`<h2>JavaScript Array length</h2>\n<p id="demo"></p>`}
    initialJs={`const fruits = ["Banana", "Orange", "Apple", "Mango"];\ndocument.getElementById("demo").innerHTML = fruits.length;`}
/>

### JavaScript Array toString()
The JavaScript method `toString()` converts an array to a string of (comma separated) array values.

<Editor 
    initialHtml={`<h2>JavaScript Array toString()</h2>\n<p id="demo"></p>`}
    initialJs={`const fruits = ["Banana", "Orange", "Apple", "Mango"];\ndocument.getElementById("demo").innerHTML = fruits.toString();`}
/>

### Popping and Pushing
When you work with arrays, it is easy to remove elements and add new elements.
This is what popping and pushing is:
Popping items **out** of an array, or pushing items **into** an array.

<Editor 
    initialHtml={`<h2>JavaScript Array po() & push()</h2>\n<p id="demo1"></p>\n<p id="demo2"></p>`}
    initialJs={`const fruits = ["Banana", "Orange", "Apple", "Mango"];\n\nfruits.pop(); // Removes Mango\ndocument.getElementById("demo1").innerHTML = fruits.toString();\n\nfruits.push("Kiwi"); // Adds Kiwi\ndocument.getElementById("demo2").innerHTML = fruits.toString();`}
/>

### Shifting Elements
Shifting is equivalent to popping, working on the first element instead of the last.
The `shift()` method removes the first array element and "shifts" all other elements to a lower index.

<Editor 
    initialHtml={`<h2>JavaScript Array shift()</h2>\n<p id="demo1"></p>\n<p id="demo2"></p>`}
    initialJs={`const fruits = ["Banana", "Orange", "Apple", "Mango"];\ndocument.getElementById("demo1").innerHTML = fruits.toString();\nfruits.shift();\ndocument.getElementById("demo2").innerHTML = fruits.toString();`}
/>
