The JavaScript Math object allows you to perform mathematical tasks on numbers.

### The Math Object
Unlike other objects, the Math object has no constructor. The Math object is static.
All methods and properties can be used without creating a Math object first.

<Editor 
    initialHtml={`<h2>JavaScript Math.PI</h2>\n<p id="demo"></p>`}
    initialJs={`document.getElementById("demo").innerHTML = Math.PI;`}
/>

### Math.round()
`Math.round(x)` returns the nearest integer.

<Editor 
    initialHtml={`<h2>Math.round()</h2>\n<p id="demo1"></p>\n<p id="demo2"></p>`}
    initialJs={`document.getElementById("demo1").innerHTML = Math.round(4.6);\ndocument.getElementById("demo2").innerHTML = Math.round(4.4);`}
/>

### Math.pow()
`Math.pow(x, y)` returns the value of x to the power of y.

<Editor 
    initialHtml={`<h2>Math.pow()</h2>\n<p id="demo"></p>`}
    initialJs={`document.getElementById("demo").innerHTML = Math.pow(8, 2);`}
/>

### Math.sqrt()
`Math.sqrt(x)` returns the square root of x.

<Editor 
    initialHtml={`<h2>Math.sqrt()</h2>\n<p id="demo"></p>`}
    initialJs={`document.getElementById("demo").innerHTML = Math.sqrt(64);`}

### Math.min() and Math.max()
`Math.min()` and `Math.max()` can be used to find the lowest or highest value in a list of arguments.

    initialJs={`document.getElementById("low").innerHTML = Math.min(0, 150, 30, 20, -8, -200);\ndocument.getElementById("high").innerHTML = Math.max(0, 150, 30, 20, -8, -200);`}
/>
