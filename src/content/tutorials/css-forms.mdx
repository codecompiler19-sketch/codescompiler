The look of an HTML form can be greatly improved with CSS.

### Styling Input Fields
Use the `width` property to determine the width of the input field.

<Editor 
    initialHtml={`<form>\n  <label for="fname">First Name</label><br>\n  <input type="text" id="fname" name="fname" class="full-width">\n</form>`}
    initialCss={`input.full-width {\n  width: 100%;\n  padding: 12px 20px;\n  margin: 8px 0;\n  box-sizing: border-box;\n}`}
/>

### Padded Inputs
Use the `padding` property to add space inside the text field.
Tip: When you have many inputs after each other, you might also want to add some `margin`.

<Editor 
    initialHtml={`<form>\n  <input type="text" placeholder="John Doe" class="my-input">\n</form>`}
    initialCss={`.my-input {\n  padding: 12px 20px;\n  margin: 8px 0;\n  box-sizing: border-box;\n  border: 2px solid #ccc;\n  border-radius: 4px;\n  background-color: #f8f8f8;\n}`}
/>

### Styling Submit Buttons
Use `background-color` to colorize the submit button and `color` to control the text color of the button.

<Editor 
    initialHtml={`<form>\n  <input type="button" class="btn" value="Submit">\n</form>`}
    initialCss={`.btn {\n  background-color: #04AA6D;\n  color: white;\n  padding: 16px 32px;\n  text-decoration: none;\n  margin: 4px 2px;\n  cursor: pointer;\n  border: none;\n  border-radius: 4px;\n}\n\n.btn:hover {\n  background-color: #038a58;\n}`}
/>
