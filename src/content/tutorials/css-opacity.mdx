The `opacity` property specifies the opacity/transparency of an element.

### Transparent Image
The opacity property can take a value from 0.0 to 1.0. The lower value, the more transparent.

<Editor 
    initialHtml={`<h2>CSS Opacity</h2>\n\n<p>Opacity 1.0 (default):</p>\n<img src="https://picsum.photos/id/237/200/300" alt="Logo">\n\n<p>Opacity 0.5:</p>\n<img src="https://picsum.photos/id/237/200/300" alt="Logo" style="opacity:0.5;">\n\n<p>Opacity 0.2:</p>\n<img src="https://picsum.photos/id/237/200/300" alt="Logo" style="opacity:0.2;">`}
/>

### Transparent Hover Effect
The opacity property is often used together with the `:hover` pseudo-class to change the opacity on mouse-over.

<Editor 
    initialHtml={`<h2>Transparent Hover</h2>\n<p>Hover over the image:</p>\n<img class="trans" src="https://picsum.photos/id/237/200/300" alt="Logo">`}
    initialCss={`.trans {\n  opacity: 0.5;\n  transition: opacity 0.3s;\n}\n\n.trans:hover {\n  opacity: 1.0;\n}`}
/>

### Transparency using RGBA
When using the `opacity` property to add transparency to the background of an element, all of its child elements inherit the same transparency. This can make the text inside a fully transparent element hard to read.
If you do not want to apply opacity to child elements, use RGBA color values instead.

<Editor 
    initialHtml={`<div class="first\">\n  <p>Opacity inherited</p>\n</div>\n\n<div class="second">\n  <p>Text is NOT transparent!</p>\n</div>`}
    initialCss={`.first {\n  background-color: blue;\n  color: white;\n  padding: 20px;\n  opacity: 0.3;\n}\n\n.second {\n  background-color: rgba(0, 0, 255, 0.3);\n  color: black;\n  padding: 20px;\n  margin-top: 20px;\n}`}
/>
