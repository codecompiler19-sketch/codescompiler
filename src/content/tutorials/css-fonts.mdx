CSS font properties define the font family, boldness, size, and the style of a text.

### CSS Font Families
In CSS, there are two types of font family names:
*   **generic family** - a group of font families with a similar look (like "Serif" or "Monospace")
*   **font family** - a specific font family (like "Times New Roman" or "Arial")

<Editor 
    initialHtml={`<p class="serif">This is a paragraph, shown in the Times New Roman font.</p>\n<p class="sansserif">This is a paragraph, shown in the Arial font.</p>\n<p class="monospace">This is a paragraph, shown in the Lucida Console font.</p>`}
    initialCss={`.serif {\n  font-family: "Times New Roman", Times, serif;\n}\n.sansserif {\n  font-family: Arial, Helvetica, sans-serif;\n}\n.monospace {\n  font-family: "Lucida Console", Courier, monospace;\n}`}
/>

### CSS Font Size
The `font-size` property sets the size of the text.

Being able to manage the text size is important in web design. However, you should not use font size adjustments to make paragraphs look like headings, or headings look like paragraphs. 

<Editor 
    initialHtml={`<h1>This is heading 1</h1>\n<h2>This is heading 2</h2>\n<p>This is a paragraph.</p>`}
    initialCss={`h1 { font-size: 40px; }\nh2 { font-size: 30px; }\np { font-size: 14px; }`}
/>

### CSS Font Style
The `font-style` property is mostly used to specify italic text.

This property has three values:
*   **normal** - The text is shown normally
*   **italic** - The text is shown in italics
*   **oblique** - The text is "leaning" (oblique is very similar to italic, but less supported)

<Editor 
    initialHtml={`<p class="normal">This is a paragraph in normal style.</p>\n<p class="italic">This is a paragraph in italic style.</p>`}
    initialCss={`p.normal { font-style: normal; }\np.italic { font-style: italic; }`}
/>
