In HTML, there are two main types of lists:
*   **Unordered lists** (`<ul>`) - the list items are marked with bullets
*   **Ordered lists** (`<ol>`) - the list items are marked with numbers or letters

The CSS list properties allow you to:
*   Set different list item markers for ordered lists
*   Set different list item markers for unordered lists
*   Set an image as the list item marker
*   Add background colors to lists and list items

### Different List Item Markers
The `list-style-type` property specifies the type of list item marker.

<Editor 
    initialHtml={`<p>Unordered lists:</p>\n<ul class="a">\n  <li>Coffee</li>\n  <li>Tea</li>\n  <li>Coca Cola</li>\n</ul>\n\n<ul class="b">\n  <li>Coffee</li>\n  <li>Tea</li>\n  <li>Coca Cola</li>\n</ul>`}
    initialCss={`ul.a {\n  list-style-type: circle;\n}\n\nul.b {\n  list-style-type: square;\n}`}
/>

### Styling Ordered Lists

<Editor 
    initialHtml={`<p>Ordered lists:</p>\n<ol class="c">\n  <li>Coffee</li>\n  <li>Tea</li>\n  <li>Coca Cola</li>\n</ol>\n\n<ol class="d">\n  <li>Coffee</li>\n  <li>Tea</li>\n  <li>Coca Cola</li>\n</ol>`}
    initialCss={`ol.c {\n  list-style-type: upper-roman;\n}\n\nol.d {\n  list-style-type: lower-alpha;\n}`}
/>

### Remove Default Settings
The `list-style-type: none` property can also be used to remove the markers/bullets. Note that the list also has default margin and padding. To remove this, add `margin: 0` and `padding: 0` to `<ul>` or `<ol>`.

<Editor 
    initialHtml={`<ul>\n  <li>Coffee</li>\n  <li>Tea</li>\n  <li>Coca Cola</li>\n</ul>`}
    initialCss={`ul {\n  list-style-type: none;\n  margin: 0;\n  padding: 0;\n  border: 1px solid #ddd;\n}\nli {\n  padding: 8px 16px;\n  border-bottom: 1px solid #ddd;\n}\nli:last-child {\n  border-bottom: none;\n}`}
/>
