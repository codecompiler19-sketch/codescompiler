JavaScript strings are used for storing and manipulating text. A JavaScript string is simply zero or more characters written inside quotes.

### String Length
To find the length of a string, use the built-in `length` property.

### Escape Character
Because strings must be written within quotes, JavaScript will misunderstand this string:

`let text = "We are the so-called "Vikings" from the north.";`

The string will be chopped to "We are the so-called ".
The solution to avoid this problem, is to use the backslash escape character. The backslash (`\\`) escape character turns special characters into string characters.

<Editor 
    initialHtml={`<h2>Escape Quotes</h2>\n<p id="demo"></p>`}
    initialJs={`let text = "We are the so-called \\"Vikings\\" from the north.";\ndocument.getElementById("demo").innerHTML = text;`}
/>

### String Methods overview
String methods help you to work with strings:
*   `String length`
*   `String slice()`
*   `String substring()`
*   `String replace()`
*   `String toUpperCase()`
*   `String toLowerCase()`
*   `String concat()`

### JavaScript Template Literals
Template Literals use back-ticks (`` ` ``) rather than the quotes (`"`) to define a string. Template literals allow variables in strings without using concatenation operators (`+`).

<Editor 
    initialHtml={`<h2>Template Literals</h2>\n<p id="demo"></p>`}
    initialJs={`let firstName = "John";\nlet lastName = "Doe";\n\n// Use back-ticks for Template Literals\nlet text = \`Welcome \${firstName}, \${lastName}!\`;\n\ndocument.getElementById("demo").innerHTML = text;`}
/>
