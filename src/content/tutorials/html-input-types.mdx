The HTML `<input>` element is the most versatile form element. Its behavior changes drastically depending on its `type` attribute.

### Common Input Types
Here are some of the most frequently used input types:
*   `text` - Default type. Single-line text input.
*   `password` - Text input where characters are masked.
*   `email` - Used for email addresses (mobile browsers show special keyboards).
*   `number` - Used for numeric input (can have limits).
*   `date` - Used for date selection.
