There are many ways to center elements in CSS.

### Center Align Elements
To horizontally center a block element (like `<div>`), use `margin: auto;`. Setting the width of the element will prevent it from stretching out to the edges of its container.

<Editor 
    initialHtml={`<h2>Center Align Elements</h2>\n<p>To horizontally center a block element, use margin: auto;</p>\n\n<div class="center">\n  <p>I am centered.</p>\n</div>`}
    initialCss={`.center {\n  margin: auto;\n  width: 50%;\n  border: 3px solid green;\n  padding: 10px;\n}`}
/>

### Center Align Text
To just center the text inside an element, use `text-align: center;`.

<Editor 
    initialHtml={`<div class="center">\n  <p>This text is centered.</p>\n</div>`}
    initialCss={`.center {\n  text-align: center;\n  border: 3px solid green;\n}`}
/>

### Center Vertically - Using Padding
There are many ways to center an element vertically in CSS. A simple solution is to use top and bottom `padding`.

<Editor 
    initialHtml={`<h2>Center Vertically</h2>\n<p>In this example, we use padding to center the div element vertically:</p>\n\n<div class="center">\n  <p>I am vertically centered.</p>\n</div>`}
    initialCss={`.center {\n  padding: 70px 0;\n  border: 3px solid green;\n  text-align: center;\n}`}
/>
