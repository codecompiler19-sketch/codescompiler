JavaScript assignment operators are used to assign values to JavaScript variables.

### The Assignment Operator (=)
The Simple Assignment Operator (`=`) assigns a value to a variable.

<Editor 
    initialHtml={`<h2>Assignment Operator</h2>\n<p id="demo"></p>`}
    initialJs={`let x = 10;\ndocument.getElementById("demo").innerHTML = "The value of x is: " + x;`}
/>

### Addition Assignment (+=)
The Addition Assignment Operator (`+=`) adds a value to a variable and assigns the new value to the variable.

<Editor 
    initialHtml={`<h2>Addition Assignment</h2>\n<p id="demo"></p>`}
    initialJs={`let x = 10;\nx += 5; // equivalent to: x = x + 5;\n\ndocument.getElementById("demo").innerHTML = "10 += 5 becomes: " + x;`}
/>

### Subtraction Assignment (-=)
The Subtraction Assignment Operator (`-=`) subtracts a value from a variable.

<Editor 
    initialHtml={`<h2>Subtraction Assignment</h2>\n<p id="demo"></p>`}
    initialJs={`let x = 10;\nx -= 5;\n\ndocument.getElementById("demo").innerHTML = "10 -= 5 becomes: " + x;`}
/>

### Multiplication and Division Assignment
The Multiplication Assignment Operator (`*=`) multiplies a variable. The Division Assignment Operator (`/=`) divides a variable.

<Editor 
    initialHtml={`<h2>Multiply and Divide</h2>\n<p id="demo1"></p>\n<p id="demo2"></p>`}
    initialJs={`let x = 10;\nx *= 5;\ndocument.getElementById("demo1").innerHTML = "10 *= 5 gives: " + x;\n\nlet y = 10;\ny /= 5;\ndocument.getElementById("demo2").innerHTML = "10 /= 5 gives: " + y;`}
/>

### List of Assignment Operators

| Operator | Example | Same As |
| :--- | :--- | :--- |
| `=` | `x = y` | `x = y` |
| `+=` | `x += y` | `x = x + y` |
| `-=` | `x -= y` | `x = x - y` |
| `*=` | `x *= y` | `x = x * y` |
| `/=` | `x /= y` | `x = x / y` |
| `%=` | `x %= y` | `x = x % y` |
| `**=` | `x **= y` | `x = x ** y` |
