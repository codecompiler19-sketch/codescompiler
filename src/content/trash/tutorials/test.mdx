## 🌐 What is CSS?

- **CSS** stands for **Cascading Style Sheets**  
- CSS is used to style and design web pages  
- It controls how HTML elements appear on screen  
- CSS helps you apply styles to multiple pages at once  
- External stylesheets are saved as `.css` files

## 💡 Why Use CSS?

CSS is essential for creating modern and attractive websites. It separates your **content (HTML)** from your **design (styling)**.

### Benefits of CSS:

- 🎨 **Easy Design Changes**  
  Update the look of an entire website by editing just one file  

- ⚡ **Faster Loading**  
  CSS files are cached by browsers, improving performance  

- 🧹 **Clean Code**  
  Keeps HTML simple and readable  

- 📱 **Responsive Design**  
  Helps create websites that work on mobile, tablet, and desktop
